== LAUNCHSPIRATION

Launchspiration generates random inspirational quotes from Launch Academy students, for future cohorts.  Originally created as part of Launch Academy's 'Ship it Saturday' Hackathon.

* Ruby 2.0.0
* Rails 4.0
* PostgreSQL

CONFIG (Create .env file with secret keys)
  export TWITTER_KEY=<TWITTER_KEY>
  export TWITTER_SECRET=<TWITTER_SECRET>

  export rails_secret_token="<RAILS_SECRET_TOKEN>"
